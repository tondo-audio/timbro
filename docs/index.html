01 — Tonal Zones

From glass to fire in a single sweep

Each zone loads a dedicated NAM profile and cabinet IR.
Adjacent zones crossfade seamlessly as you turn the dial.

01
Clean
Dial 0 — 2
Crystal clear. Uncolored headroom for dynamics and articulation.
02

03
Crunch
Dial 4 — 6
Classic midgain. Responsive crunch that cleans up with your volume knob.
04
Drive
Dial 6 — 8
Saturated gain. Thick, compressed drive with sustain for days.
05
Lead
Dial 8 — 10
Full saturation. Liquid high-gain for soaring leads and heavy riffs.

05 — Open Source

Build it yourself

Everything you need to compile, modify, and ship your own version.

Terminal
# Clone
$ git clone https://github.com/tondo-audio/timbro.git
$ cd timbro

# Configure & Build
$ cmake -B build -DCMAKE_BUILD_TYPE=Release
$ cmake --build build --config Release

# AU plugin auto-installs to
# ~/Library/Audio/Plug-Ins/Components/

# Allow unsigned AUs in Logic Pro
$ defaults write com.apple.Logic10 \
    DoNotValidateAudioUnits -bool YES
